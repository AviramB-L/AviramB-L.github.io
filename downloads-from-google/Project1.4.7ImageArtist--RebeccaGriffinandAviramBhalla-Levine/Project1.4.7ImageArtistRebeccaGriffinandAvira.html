Log:

Day and Newly Modified Image

Accomplished

3/13/17

Began brainstorming what to do. Eventually decided try to change color pictures into black and white.

Version 0

There was success in our decoloration, however there were colors where it should have been black. While Version 0 was interesting to look at, it was not what we wanted.

Version 1

Version 1 is almost the opposite of Version 0, in that we now have too much disappearing. Will fine-tune tomorow.

3/14/17

Version 2, Version 3, and Version 4 were completed.

Version 2

We were able to define the image in black better than version 1. Adds whole image directory, not just one file at a time.  

Version 3 

Not a huge difference between Version 2 and Version 3, just making some code more efficient.

Version 4

Not a huge difference, just some slight tweaking with docstrings and comments.

Note: Cat image needs fixing, too many pixels still.

3/15/17

Both of us were sick, and didn’t get time to work at home

3/16/17

(trumpet still

looks the same)

Version 5 and Version 6 were completed.

Version 5

This version helped refine the changes, so that there were less miscellaneous pixels muddying up the image.  

Version 6

Minor changes.

Will display cat image from now on due to importance of changes.

3/20/17

Created the documentation, conferred with the people next to us about our code and things we could do to improve.

Version 7

Rewrote functions to remove extraneous code. Added the filling method, but has some bugs that erase the image.

3/21/17

We continued work on

Version 8 and fixed the image-erasing bug.

The images are saved as they go, not all at once like in previous programs.

Worked on Version 9 and Version 10 at home, succeeded in removing extraneous dots on many of the real-life images, such as the cat and giraffe image.

3/22/17

Worked on Version 11 and Version 12 to ensure that all pixels in the trace would be black and allow for the background to be any color desired or completely transparent.

Version 13 now allows for the background color of each image to be specified individually.
